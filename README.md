\# AndroidTrivia 单元测试项目



\## 项目介绍

AndroidTrivia 是一个问答游戏应用，本项目为它添加完整的单元测试。



\## 测试环境

\- Android Studio 4.0.1

\- Gradle 6.1.1

\- JDK 8

\- JUnit 4.13.2 + Robolectric 4.5.1 + MockK 1.10.6
